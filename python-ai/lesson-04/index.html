LIST INDEX

list 的每个元素都有位置编号

数据

numbers = [1, 3, 5, 7, 9]

索引

numbers[0] 是 1，numbers[4] 是 9。

Python 从 0 开始计数。

常见错误：最后一个元素不是 numbers[5]，而是 numbers[4]。
用手指表格位置，让学生先预测 numbers[2] 是什么。

LOOP TWO WAYS

遍历 list 有两种常用方式

for-each

for number in numbers:
    print(number)

适合只读取每个值。

index loop

for i in range(len(numbers)):
    print(numbers[i])

适合需要位置或要修改元素。

强调 for number in numbers 修改的是临时变量，不会自动改原 list。

COMMON TRAP

for-each 里的变量不是原列表格子

看起来像修改

for score in scores:
    score += 5

只改了临时变量 score。

真正修改 list

for i in range(len(scores)):
    scores[i] += 5

通过 index 写回原来的位置。

判断标准：等号左边是不是 scores[i]？
这部分来自原课件中 number *= 2 但原 list 不变的动画。

留言任务：写一句：什么时候用 for-each，什么时候用 index loop？
商博老师 · 朗读 / 解释Professor Shangbo · Quote / Explanation
list 让我们把很多值放在一起；index 让我们精确地指向其中一个位置。 A list lets us keep many values together; an index lets us point to one exact position.

只读取时可以 for-each；需要改原列表、需要位置、需要二维表格时，index loop 更清楚。 Use for-each when you only read values. Use an index loop when position or updating the original list matters.
