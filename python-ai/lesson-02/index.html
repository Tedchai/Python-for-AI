IF STATEMENT

if 让程序选择是否进入一个代码块

1
Evaluate条件表达式得到 True 或 False
2
Enter or skipTrue 执行缩进代码块，False 跳过
3
Continue代码块结束后继续往下走
if number > 0:
    print("positive")
print("always runs")
强调缩进是 Python 的结构，不是排版。让学生用手指出哪些行属于 if。

在 Python 里，哪件事决定一行代码是否属于 if 代码块？

Python 用缩进表示代码块。缩进不同，程序逻辑就可能完全不同。

可以现场把 print('always runs') 缩进进去，再让学生预测输出变化。
IF / ELIF / ELSE

多分支判断：只选择一条路径

二分支

  • if 条件成立：走第一条路
  • 否则 else：走备用路线
  • 适合 pass / fail、yes / no

多分支

  • 先检查 if
  • 再逐个检查 elif
  • 都不满足时走 else
  • 适合 positive / negative / zero
口诀：从上到下检查，遇到第一个 True 就执行，然后离开整组分支。
用 number = 5、-2、0 三个例子，让学生分别说出路径。

FOR LOOP

for loop 适合已知次数或已知序列

range(5)

0, 1, 2, 3, 4

range(-10, 11)

-10 到 10，包含 -10，不包含 11

range(0, 10, 2)

0, 2, 4, 6, 8

stop 不包含：这是 range 最常见的边界错误。
让学生先写出 range(-10, 11) 的第一个和最后一个数字，再运行验证。
RANGE

Quick check · tap an answer

list(range(0, 5)) 的结果是什么？

range 的 stop 是 exclusive，不包含 5。

如果学生混淆，可以和切片、半开区间做类比。
WHILE LOOP

while 适合“不知道要重复几次”的任务

for

  • 遍历已知序列
  • 通常知道循环范围
  • 例如检查 scores 列表

while

  • 只要条件为 True 就继续
  • 适合输入验证
  • 必须设计终止条件
count = 3
while count > 0:
    print(count)
    count -= 1
无限循环是重点风险：每次讲 while 都要问，这个条件什么时候会变成 False？
BREAK / CONTINUE

两个词，控制循环的节奏

break

提前结束整个循环。常用于“已经找到了，不用继续”。

continue

跳过当前这一轮剩下的代码，直接进入下一轮。

课堂问法：它是离开整个循环，还是只跳过这一次？
用 0-9 只打印奇数的例子解释 continue；用找到第一个合格分数解释 break。
