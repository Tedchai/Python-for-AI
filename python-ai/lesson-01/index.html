GRADING

强调变量不是数学里的未知数，而是程序里的命名存储。
DATA TYPES

Quick check · tap an answer

如果代码是 age = input("Age: ")，那么 age 默认是什么类型？

input() 读到的内容默认是字符串 str。要做数学计算，需要 int(age) 或 float(age)。

这道题为 mini profile 中的 int(input(...)) 做铺垫。

DEBUGGING MINDSET

报错不是终点，是线索

先读最后一行

通常最后一行告诉你错误类型和最直接原因。

再看行号

定位是哪一行触发了问题，不要全篇乱改。

小步重跑

一次只改一个地方，确认修复是否有效。

能解释再提交

修好代码后，说清楚为什么错、怎么改。

可以现场故意制造一个 ValueError 或 SyntaxError，让学生按这四步找线索。

商博老师 · 朗读 / 解释Professor Shangbo · Quote / Explanation
第一段代码不需要很长；它只需要证明一件事：你可以让计算机听懂你的指令。 Your first program does not need to be long; it only needs to prove one thing: you can make the computer understand your instructions.

今天的重点不是背语法，而是建立信心：能运行、能修改、能解释，就是进入 Python 的第一步。 The point today is not memorizing syntax. It is building confidence: run it, change it, and explain it.
