MISTAKE CLINIC

Common errors to expect

1

Missing quotes

Text values need quotation marks.

2

Variable typo

topic and topics are different names.

3

Not rerunning

After edits, run the notebook from top to bottom.

Normalize errors and teach a response routine.
DEBUG ROUTINE

Read, locate, isolate, rerun

1ReadRead the last error line.
2LocateFind the line number.
3IsolateTest the smallest piece.
4RerunRun top to bottom.
Repeat this routine every week.

AI AS STUDY HELPER

Use AI to learn Python, not to skip Python

Good prompts

  • Explain this error like I am a beginner.
  • Give me a similar practice problem.
  • Ask me hints before giving the answer.

Avoid

  • Write the whole homework for me.
  • Rewrite someone else's code so I can submit it.
  • Give me code I cannot explain.
Keep AI use aligned with learning.
